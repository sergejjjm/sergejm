More about this project

This is a company website with a blog ( Webflow's CMS). One page features seven attractively designed and animated sections, as well as a page labeled "blog."

About the project's UX/UI design.

A hero section, two sections with sliders, a blog section, and a footer are all on one of the two pages of this website. All the blogs are on the second page called "blog". In addition to flawless page animations, this project offers excellent hover animations. The site is fully responsive on all screen sizes.

The website's symbols and pictures are both SEO-friendly. With the help of Webflow, it loads rapidly on all devices, is secure, and is relevant without requiring any coding.
